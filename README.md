# PyramidMeshSolver

Програмная реализация решения эллиптического дифференционального уравнения на пирамидальной конечно-элементной сетке методом конечных элементов. В качестве базисных функций используются трилинейные, эрмитовы триквадратичные и эрмитовы трикубические. Решение СЛАУ с помощью метода сопряженных градиентов.

## Technologies Used

- С++
- CMake
- Python
- Clang-tidy

## Installation

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Sharero/PyramidMeshSolver.git
2. Перейдите в папку проекта:
   ```bash
   cd PyramidMeshSolver
3. Создайте и перейдите в папку для сборки проекта:
   ```bash
   mkdir /build && cd /build
4. Запустите сборку с помощью Ninja
   ```bash
   cmake -DCMAKE_BUILD_TYPE=Release -G Ninja ..
5. Выполните запуск проекта
   ```bash
   ninja

## Input data

Входные данные находятся в **data/input**:
- boundaries.txt - первое число количество узлов с краевыми условиями; индексы узлов в глобальном нумерации
- gridInfo.txt - 3 строчки содержащих по 2 числа: минимальное и максимальное значение по осям х, у, z; шаг по х, шаг по y, шаг по z

## Output data

Выходные данные находятся в **data/output**:
- nodes.txt - координаты x,y,z точек в глобальной нумерации
- elements.txt - индексы точек для всех пирамидальных элементов
- result.txt - таблица из 6 столбцов: координата по х, y, z, значение МКЭ в этой точке, истинной решение в этой точке, погрешность решения
